# DialogueNodeEditor
基于GraphView制作的可视化对话节点编辑器  
想要下载的去master分支下载  
该编辑器使用了github用户BusStopStudios分享的适用于 Unity 的基于节点的可视化图形系统框架  
原框架：https://github.com/BusStopStudios/Unity-Visual-Graph-Editor
